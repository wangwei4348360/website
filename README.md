# 通用官方网站

一个现代化、响应式的通用官方网站模板，具有美观的设计和丰富的交互功能。

## 🌟 特性

### 设计特性
- **现代化UI设计** - 采用最新的设计趋势，界面简洁美观
- **响应式布局** - 完美适配桌面、平板和移动设备
- **渐变背景** - 使用优雅的渐变色彩搭配
- **动画效果** - 丰富的CSS动画和过渡效果
- **图标支持** - 集成Font Awesome图标库

### 功能特性
- **固定导航栏** - 带有毛玻璃效果的固定导航
- **平滑滚动** - 页面内锚点平滑滚动
- **移动端菜单** - 响应式汉堡菜单
- **表单验证** - 联系表单的完整验证功能
- **通知系统** - 美观的消息通知提示
- **数字动画** - 统计数据的动态计数效果
- **滚动动画** - 元素进入视口时的淡入动画

### 技术特性
- **纯HTML/CSS/JS** - 无需框架，轻量级
- **语义化HTML** - 良好的SEO和可访问性
- **模块化CSS** - 清晰的样式组织结构
- **性能优化** - 防抖处理，流畅的用户体验

   ```bash
   # 将所有文件下载到本地目录
   ```

2. **打开网站**

## 🎨 自定义指南

### 修改颜色主题
在 `styles.css` 中修改以下CSS变量：
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #667eea;
    --text-color: #333;
    --background-color: #f8fafc;
}
```

### 添加新页面
1. 创建新的HTML文件
2. 复制导航栏和页脚结构
3. 在导航菜单中添加链接

### 修改公司信息
在 `index.html` 中搜索并替换：
- 公司名称：`CompanyName`
- 联系信息：电话、邮箱、地址
- 社交媒体链接
